

# Internet Download Manager Activator - Rem Version

This is an IDM activator using CMD and PowerShell. It safely edits the IDM registry. This script is 100% clean, so you can trust Rem, right? If you're unsure, you can read the script, Onii-chan!

## How to Activate IDM?

* 1st - Open PowerShell as Administrator.
* 2nd - Run the command below:
```bash
irm https://raw.githubusercontent.com/PhitzZ/REM_IDMActivator/main/IDMrem.cmd -OutFile "$env:TEMP\IDMrem.cmd"; Start-Process cmd -ArgumentList "/c `"$env:TEMP\IDMrem.cmd`"" -Verb RunAs
```
* Last - Choose your option, and voilà! free IDM for a lifetime!
